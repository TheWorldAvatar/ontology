# OntoSyn ontology export

The Turtle file in this folder is a serialized export of the OntoSyn ontology.

## TWA Object-graph-mapper

This ontology was defined in Python using the TWA OGM (object-graph-mapper), see https://github.com/TheWorldAvatar/baselib/tree/main/python_wrapper

## Ontology ground truth

The ontology ground truth is the corresponding Python file in the MOP-specific repository, see https://github.com/TheWorldAvatar/MOPTools/blob/main/MOP_Literature_Extraction/OntoSyn_ontology.py
